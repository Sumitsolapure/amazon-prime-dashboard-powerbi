# 📊 Prime Video Content Analysis Dashboard (Power BI)

## 🚀 Overview
Built an interactive Power BI dashboard to analyze Prime Video content based on type, genre, ratings, and release trends.

---

## 🛠 Tools Used
- Power BI  
- Excel  
- Power Query  
- DAX  

---

## 📊 Dashboard Highlights
- Total Titles: 9687  
- Movies vs TV Shows distribution  
- Genre-wise content analysis  
- Rating-wise distribution  
- Year-wise content growth  

---

## 📈 Key Analysis
- Most content is Movies compared to TV Shows  
- Popular genres include Drama, Suspense, and Western  
- Majority content falls under “R” and “PG-13” ratings  
- Rapid growth in content after 2015  

---

## 💡 Key Insights
- Movies dominate the platform (~80%)  
- Content production increased significantly in recent years  
- Certain genres consistently have higher content volume
